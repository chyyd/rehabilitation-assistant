# 康复科助手 - 用户指南

## 安装

### 1. Python环境准备
确保安装了 Python 3.10 或更高版本：
```bash
python --version
```

### 2. 安装依赖
```bash
pip install -r requirements.txt
```

### 3. 配置API密钥

编辑 `config.json` 或创建 `.env` 文件：
```env
MODELSCOPE_API_KEY=your_api_key
DEEPSEEK_API_KEY=your_api_key
KIMI_API_KEY=your_api_key
```

## 使用

### 启动应用
```bash
python main.py
```

### 创建患者
1. 点击"➕新患者"按钮
2. 输入住院号
3. 粘贴首次病程记录
4. 确认提取的信息
5. 保存

### 生成病程记录
1. 从左侧列表选择患者
2. 输入当日情况
3. 点击"✨ AI生成"
4. 查看并编辑生成的记录
5. 保存或导出

## 功能说明

### AI病程记录生成
- 支持多种记录类型（住院医师/主治医师/主任医师查房记录）
- 自动结合患者历史和今日情况
- 符合中医病历规范

### 智能提醒系统
- 自动提醒查房记录
- 90天住院限制提醒
- 化验复查提醒

### 患者管理
- 从首次病程记录自动提取信息
- 支持患者信息编辑
- 出院管理

### 知识库
- 支持PDF/EPUB/Word文档导入
- 语义检索相关内容
- 辅助病程记录生成

## 常见问题

### Q: AI生成失败怎么办？
A: 检查API密钥是否配置正确，确保网络连接正常。

### Q: 如何添加自定义模板？
A: 在数据库中添加模板记录，或通过界面创建（功能开发中）。

### Q: 支持哪些文档格式？
A: 支持PDF、EPUB、Word (.doc/.docx)和纯文本文件。

## 快捷键
- `Ctrl+N`: 创建新患者
- `Ctrl+S`: 保存
- `Ctrl+Enter`: AI生成
