## 🎮 体验建议

### 第一次使用？按这个顺序体验：

1. **查看患者列表**（左侧）
   - 注意不同的优先级颜色（红/黄/绿）
   - 点击不同患者卡片

2. **体验AI生成功能**（中间）
   - 在"当日情况"输入框输入：
     ```
     患者右上肢肌力较前改善，可完成抓握动作，继续康复训练
     ```
   - 点击"✨ AI生成"按钮
   - 等待1.5秒查看生成结果

3. **尝试快速模板**（右侧）
   - 点击"诊断模板"下拉框
   - 选择任意诊断
   - 点击"常用短语"快速插入

4. **创建新患者**
   - 点击顶部"➕新患者"按钮
   - 体验3步创建流程

5. **查看设置**
   - 点击顶部"⚙️"按钮
   - 切换不同标签页

### ⌨️ 键盘快捷键测试

- `Ctrl + N`：打开新患者窗口
- `Ctrl + S`：保存病程记录
- `Ctrl + Enter`：生成病程记录
- `ESC`：关闭弹窗

### 常见问题

**Q: 页面显示空白？**
A: 请使用Chrome或Edge浏览器，不要用IE。

**Q: 样式显示不正常？**
A: 确保 `styles.css` 和 `index.html` 在同一目录。

**Q: 点击没反应？**
A: 检查浏览器控制台（F12）查看错误信息。
