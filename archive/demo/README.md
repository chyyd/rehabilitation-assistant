# 康复科助手 - 交互式界面演示

## 📋 项目简介

这是一个为中医院康复科设计的病历与事务管理系统的**交互式HTML演示**，展示了完整的iOS风格现代化界面设计。

## ✨ 功能展示

### 1. 智能患者管理
- ✅ 患者列表按优先级排序（紧急/高/正常）
- ✅ 一键选择患者，查看详细信息
- ✅ 自动显示患者住院天数、诊断等关键信息

### 2. AI病程记录生成
- ✅ 输入当日情况，AI智能生成病程记录
- ✅ 支持编辑和微调生成的内容
- ✅ 一键保存和导出为TXT文件

### 3. 智能提醒系统
- 🔔 住院天数预警（80天、90天提醒）
- 📝 查房记录提醒
- 🔬 化验检查提醒

### 4. 快速模板系统
- 📋 诊断模板快速插入
- 💬 常用短语一键添加
- 📜 历史记录快速加载

### 5. 新患者创建流程
1️⃣ 输入住院号
2️⃣ 粘贴首次病程记录
3️⃣ AI自动提取信息
4️⃣ 人工确认保存

### 6. 系统设置
- ⚙️ AI服务配置（多服务切换）
- 👨‍⚕️ 医师信息管理
- 📚 知识库管理

## 🚀 快速开始

### 方法1：直接打开（推荐）

双击 `demo/index.html` 文件，在浏览器中打开即可。

### 方法2：使用本地服务器

```bash
# 进入demo目录
cd demo

# 使用Python启动服务器
python -m http.server 8000

# 在浏览器访问
# http://localhost:8000
```

## 💻 系统要求

- **浏览器**：Chrome、Edge、Firefox、Safari（现代浏览器）
- **分辨率**：推荐 1440x900 或更高
- **无需安装**：纯HTML/CSS/JavaScript，无需任何依赖

## 🎮 交互演示指南

### 基础操作

1. **选择患者**
   - 点击左侧患者列表中的任意卡片
   - 查看患者详细信息更新

2. **生成病程记录**
   - 在"当日情况"输入框输入文本
   - 点击"AI生成"按钮
   - 查看生成的病程记录
   - 可直接编辑预览内容

3. **使用模板**
   - 右侧快速模板区域
   - 选择诊断模板、处理意见、宣教内容
   - 点击常用短语快速插入

4. **创建新患者**
   - 点击顶部"➕新患者"按钮
   - 按提示完成3步创建流程
   - 体验AI信息提取功能

5. **查看设置**
   - 点击顶部"⚙️"按钮
   - 切换不同标签页查看配置

### 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl/Cmd + N` | 创建新患者 |
| `Ctrl/Cmd + S` | 保存病程记录 |
| `Ctrl/Cmd + Enter` | AI生成病程记录 |
| `ESC` | 关闭模态框 |
